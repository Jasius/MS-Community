# Windows Keyboard shortcuts

Mastering keyboard shortcuts can significantly enhance your productivity on Windows.

This article will introduce you to the essential shortcuts that will help you navigate the new operating system with ease.

## General shortcuts (applies to all versions of Windows)

| Combinations                                                                                | Actions                                                                                         |
| ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| <kbd>Ctrl</kbd> + <kbd>X</kbd>                                                              | Cut the selected element.                                                                       |
| <kbd>Ctrl</kbd> + <kbd>C</kbd>                                                              | Copy the selected element.                                                                      |
| <kbd>Ctrl</kbd> + <kbd>V</kbd>                                                              | Paste the selected element.                                                                     |
| <kbd>Ctrl</kbd> + <kbd>Z</kbd>                                                              | Undo an action.                                                                                 |
| <kbd>Alt</kbd> + <kbd>Tab</kbd>                                                             | Switch between open apps.                                                                       |
| <kbd>Alt</kbd> + <kbd>F4</kbd>                                                              | Closes the active app. If you're on desktop, the shutdown dialog will appear.                   |
| <kbd>Win</kbd> + <kbd>L</kbd>                                                               | Locks your PC.                                                                                  |
| <kbd>Win</kbd> + <kbd>D</kbd>                                                               | Displays and hides app windows.                                                                 |
| <kbd>F2</kbd>                                                                               | Renames the selected item.                                                                      |
| <kbd>F3</kbd>                                                                               | Searches for a file or folder in File Explorer.                                                 |
| <kbd>F4</kbd>                                                                               | Displays the address bar in File Explorer.                                                      |
| <kbd>F5</kbd>                                                                               | Refreshes the active window.                                                                    |
| <kbd>F6</kbd>                                                                               | Cycles through screen elements in a window or on desktop.                                       |
| <kbd>F10</kbd>                                                                              | Activates the Menu bar in the active app.                                                       |
| <kbd>Alt</kbd> + <kbd>F8</kbd>                                                              | Shows your password on the sign-in screen.                                                      |
| <kbd>Alt</kbd> + <kbd>Esc</kbd>                                                             | Cycles through items in the order in which they were opened.                                    |
| <kbd>Alt</kbd> + Underlined letter                                                          | Performs the command on that letter.                                                            |
| <kbd>Alt</kbd> + <kbd>Enter</kbd>Enter                                                      | Displays properties for the selected item.                                                      |
| <kbd>Alt</kbd> + <kbd>Spacebar</kbd>                                                        | Opens the shortcut menu for the active window.                                                  |
| <kbd>Alt</kbd> + <kbd>←</kbd>                                                               | Go back.                                                                                        |
| <kbd>Alt</kbd> + <kbd>→</kbd>                                                               | Go forward.                                                                                     |
| <kbd>Alt</kbd> + <kbd>Page Up</kbd>                                                         | Moves up one screen.                                                                            |
| <kbd>Alt</kbd> + <kbd>Page Down</kbd>                                                       | Moves down one screen.                                                                          |
| <kbd>Ctrl</kbd> + <kbd>F4</kbd>F4                                                           | Closes the active window.                                                                       |
| <kbd>Ctrl</kbd> + <kbd>A</kbd>A                                                             | Selects all elements in a document or window.                                                   |
| <kbd>Ctrl</kbd> + <kbd>D</kbd> or <kbd>Del</kbd>                                            | Deletes the selected item an move it.                                                           |
| <kbd>Ctrl</kbd> + <kbd>E</kbd>                                                              | Opens Search option. (if available)                                                             |
| <kbd>Ctrl</kbd> + <kbd>R</kbd> or <kbd>F5</kbd>                                             | Refreshes the active window.                                                                    |
| <kbd>Ctrl</kbd> + <kbd>Y</kbd>                                                              | Redo an action.                                                                                 |
| <kbd>Ctrl</kbd> + <kbd>→</kbd>                                                              | Moves the cursor to the beginning of the next word.                                             |
| <kbd>Ctrl</kbd> + <kbd>←</kbd>                                                              | Moves the cursor to the beginning of the previous word.                                         |
| <kbd>Ctrl</kbd> + <kbd>↓</kbd>                                                              | Moves the cursor to the beginning of the next paragraph.                                        |
| <kbd>Ctrl</kbd> + <kbd>↑</kbd>                                                              | Moves the cursor to the beginning of the previous paragraph.                                    |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Tab</kbd>                                           | Switch between opened apps with arrow keys. (You can let go of the <kbd>Alt</kbd> and tab keys) |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>↑</kbd> <kbd>↓</kbd> <kbd>→</kbd> <kbd>←</kbd>     | When a group or tile is selected on the Start menu, move it in the specified direction.         |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>↑</kbd> <kbd>↓</kbd> <kbd>→</kbd> <kbd>←</kbd>    | When a tile is selected on the Start menu, drag it onto another tile to create a folder.        |
| <kbd>Ctrl</kbd> + <kbd>↑</kbd> <kbd>↓</kbd> <kbd>→</kbd> <kbd>←</kbd> + <kbd>Spacebar</kbd> | Select multiple individual items within a window or on the desktop.                             |
| <kbd>Ctrl</kbd> + <kbd>Esc</kbd>                                                            | Opens Start Menu.                                                                               |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Esc</kbd>                                         | Opens Task Manager.                                                                             |
| <kbd>Alt</kbd> + <kbd>Shift</kbd>                                                          | Changes the keyboard layout when multiple layouts are available.                                |
| <kbd>Shift</kbd> + <kbd>F10</kbd>                                                           | Shows the context menu for the selected item.                                                   |
| <kbd>Shift</kbd> with any arrow keys                                                        | Select multiple items within a window or on the desktop, or highlight text in a document.       |
| <kbd>Shift</kbd> + <kbd>Del</kbd>                                                           | Permanently delete the selected item without sending it to the Recycle Bin.                     |
